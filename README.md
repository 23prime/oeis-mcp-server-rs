## Development

### Pre-requirements

- [Taskfile](https://taskfile.dev)
- [mise](https://mise.jdx.dev)

### Run application

1. Setup

    ```sh
    task setup
    ```

2. Run app

    ```sh
    task rs:dev
    ```

3. Run integration tests at another terminal

    ```sh
    task t:check
    ```
